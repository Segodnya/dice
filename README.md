# Dice Roller - Бросаем кубики

Веб-приложение, которое пригодится, когда кубик закатился под диван.

## Обзор

- mobile-friendly интерфейс;
- по тапу на игровом поле (кроме тапа по кубику) выбрасывается 6d-кубик;
